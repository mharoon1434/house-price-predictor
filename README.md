<h1 align="center">🏠 House Price Prediction Using Machine Learning</h1>

<p align="center">
A Machine Learning project that predicts house prices using regression algorithms and AI concepts.
</p>

---

<h2>Project Overview</h2>

<p>
This project focuses on predicting house prices using <b>Machine Learning (ML)</b> and 
<b>Artificial Intelligence (AI)</b> concepts. The model is trained on housing datasets 
containing features such as location, area, number of bedrooms, bathrooms, and other 
property-related attributes.
</p>

<p>
The main objective is to build a regression model that can estimate house prices accurately.
</p>

<h2>Regression Formula</h2>

<p align="center">

\[
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n
\]

</p>

<p>
Where:
</p>

<ul>
  <li><b>y</b> = Predicted House Price</li>
  <li><b>x</b> = Input Features</li>
  <li><b>β</b> = Regression Coefficients</li>
</ul>

<h2>Installation</h2>
